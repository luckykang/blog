### 一.go_path结构

├── bin   存放编译后的二进制文件

├── pkg   存放编译后的库文件 

├── src   存放源码文件 

│   └──网站域名└── 作者/机构└──项目名

### 二.hello word代码结构
	//包，表明代码所在的模块，package必须为main函数
	package main 

	//fmt用于打印数据
	import (
		"fmt"   
	)

	//功能实现，必须为main函数
	func main() {
		fmt.Println("hello world")
	}

### 三.Print、Println、Printf区别
- 1.一次输入多个值的时候，Println中间有空格，Print没有

- 2.Println会自动换行，print不会
 
- 3.Printf用于格式化输出

		func main() {
			fmt.Print("ni","hao","ya")
			fmt.Print("hello golang")
			fmt.Println("hello","world","k")
			fmt.Println("hello golang")

			// go语言中变量定义了必须使用
			var a = 0  
			var b = "NN"
			fmt.Println(a)
			fmt.Println("%v",b)

			//也可以用类型推导方式定义变量
			a := 10
			fmt.Printf("a=%v a的类型是%T",a,a)
		}
